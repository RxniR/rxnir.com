Learn how to use and configure my Minecraft datapacks

This documentation page is only used to show how to download and use my projects, if you want to publish your own, visit the Deveolper Documentation page.

Introduction

Welcome to the documentation for RxniR's Minecraft Datapacks. This guide will walk you through the installation process, key features, and how to customize the datapacks to fit your needs.

Installation Instructions

Follow these steps to install a datapack:

  1. Download the desired datapack from Modrinth, CurseForge or from the Projects Page.
  2. Navigate to your Minecraft world save folder. For example:

    Minecraft

    %appdata%/.minecraft/saves/YourWorldName/

    CurseForge

    curseforge/minecraft/Instances/YourInstancesName/saves/YourWorldName

    Modrinth

    %appdata%/ModrinthApp/profiles/YourProfileName/saves/YourWorldName
    3. the downloaded datapack `.zip` file into the `datapacks` folder of your world save.
  3. You can also use a loader, like Paxi, Open Loader or Global Packs
  4. Start or reload the world using the command: 
    /reload

How to Use

Once installed, you can activate the datapack by following the in-game instructions:

    Use the command /datapack list to ensure the datapack is loaded.

    Now that the datapack is set up, here are some things you can do, with your datapack.

  • Get a cutting board.
  • Get a knife of any rarity.
  • Place your modded log onto the cutting board.
  • Right click the cutting board.
  • And thats it, then you'll get a piece of bark.

    • For more information regarding the farmer's delight mod, check out the Official Wiki.

Additional Notes

Here are some additional tips for working with datapacks:

  • Always back up your world before adding or removing datapacks.
  • If a datapack isn't working as expected, check the Minecraft log for errors.
  • For support or suggestions, feel free to contact me via rxnirdatapack.dev@gmail.com.
